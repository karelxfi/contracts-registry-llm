DeFi Contracts Registry API

Static JSON API for querying DeFi protocol contract addresses across multiple chains

https://karelxfi.github.io/contracts-registry-llm

Endpoints

Get all protocols
GET /data/generated/protocols.json
Returns metadata for all 4,442 protocols
Response
Get protocols by chain
GET /data/generated/by-chain/{chain}.json
Example: /data/generated/by-chain/ethereum.json
Response
Get protocols by category
GET /data/generated/by-category/{category}.json
Example: /data/generated/by-category/lending.json
Response
Reverse lookup by address
GET /data/generated/indexes/by-address.json
Search for protocols by contract address
Response
Get populated protocols
GET /data/populated-protocols.json
Quick lookup for protocols with addresses
Response
Get specific protocol
GET /data/sources/protocols/{protocol}/{version}.json
Example: /data/sources/protocols/aave-v3/aave-v3.json
Response
Get chain metadata
GET /data/sources/chains/{chain}.json
Example: /data/sources/chains/ethereum.json
Response

Example Usage

# Get all Ethereum protocols
curl https://karelxfi.github.io/contracts-registry-llm/data/generated/by-chain/ethereum.json

# Get Aave V3 data
curl https://karelxfi.github.io/contracts-registry-llm/data/sources/protocols/aave-v3/aave-v3.json

# Get all lending protocols
curl https://karelxfi.github.io/contracts-registry-llm/data/generated/by-category/lending.json
